## 소개

> 불법 벌목과 산불로부터 숲을 지키기 위한 솔루션

> 선린인터넷등학교 IoT 아이디어 경진대회 금상 수상작

## 기능 소개

- **앱**: ThingSpeak API에서 IoT 디바이스의 정보와 상태를 가져와 Google Map에 표시
- **바람**: 바람 상태 확인
